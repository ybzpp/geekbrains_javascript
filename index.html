Задание №1

var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2
d = b++; alert(d); // 1
c = (2+ ++a); alert(c); // 5
d = (2+ b++); alert(d); // 4
alert(a); // 3
alert(b); // 3

Постфиксная форма i++ отличается от префиксной ++i тем, что возвращает старое значение,
бывшее до увеличения.

это не объект, а отдельный тип данных.
